●概要
　MLTをはじめとした(´д｀)Editで読み込み可能なファイルを
　ページごとにPNGファイルへと変換するためのコンバータです。

　MLTと名付いていますが、AST、ASDなどのデータも入力可能です。


●動作環境
　Microsoft Windows 10 以上
　Microsoft Internet Explorer 4 以上

　なお、動作環境を満たしている場合でも動作しない場合があるかもしれません。


●基本的な使い方
　MLT2PNG.exe にpngにしたいファイルをドラッグ･アンド･ドロップしてください。

　変換したファイルは、元ファイルと同一ディレクトリに変換元のファイル名のディレクトリが生成され、
　そのディレクトリ内に「ページ番号.png」として保存されます。
　すでにファイルが存在する場合、上書きされますので注意してください。


●ライセンス
　・生成物について
　　生成物の著作権、つまりこのツールによって
　　変換されたファイルの著作権は、元のファイルの著作者に在るものとします。
