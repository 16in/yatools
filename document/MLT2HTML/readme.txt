●概要
　MLTをはじめとした(´д｀)Editで読み込み可能なファイルを
　HTMLファイルへと変換するためのコンバータです。

　MLTと名付いていますが、AST、ASDなどのデータも入力可能です。


●動作環境
　Microsoft Windows XP 以上
　Microsoft Internet Explorer 4 以上

　なお、動作環境を満たしている場合でも動作しない場合があるかもしれません。


●基本的な使い方
　MLT2HTML.exe にhtmlにしたいファイルをドラッグ･アンド･ドロップしてください。

　変換したファイルは、元ファイルと同一ディレクトリに生成されます。
　同一ファイル名のものが存在する場合、上書きされますので注意してください。


●使い方(コマンドライン)
　MLT2HTML [-o 出力先ディレクトリ] [-d 変数名 置き換え文字列] 入力ファイル ...

　　-oオプション
　　　出力先に指定されたディレクトリにhtmlファイルを吐き出します。
　　　出力先が不正であった場合、このオプションは無視されます。

　　　また、複数指定された場合は有効な出力先のうち、一番最後に指定されたものが反映されます。

　　入力ファイル...
　　　(´д｀)Editで読み込み可能なファイルを指定してください。
　　　複数のファイルを同時に指定可能で
　　　-oオプションで指定したディレクトリ、或いは元ファイルと同じディレクトリに
　　　htmlファイルを出力します。


●libM2Hの使い方
　libM2HはMLT2HTMLの機能を外部に提供する動的リンクライブラリです。
　テンプレートをセットアップし、実行関数に入力ファイルパスか
　変換元文字列を渡すことによって、変換処理の実行をMLT2HTMLと共通化することが出来ます。

　前準備.
　　m2h.hをインクルードし、libM2H.libをリンクして
　　libM2H.dllをパスが通っているディレクトリに配置します。


　1.テンプレートのセットアップ
　　M2H_SetupTemplate関数にtemplateディレクトリの一つ上のディレクトリを設定し
　　テンプレートディレクトリ内のテンプレートをセットアップします。

　　なお、ここに指定するディレクトリは
　　例えばc:/work/m2h/templateディレクトリがテンプレートディレクトリである場合
　　M2H_SetupTemplate( "c:/work/m2h" );
　　として呼び出して下さい。

　　現状、テンプレートディレクトリ名がtemplateであるのは仕様です。
　　将来的にテンプレートディレクトリ名を指定できるように拡張する予定です。


　2.HTML生成ルーチンを実行
　　M2H_Run関数、あるいはM2H_RunString関数を使用して、HTMLファイルを生成します。

　　M2H_RunとM2H_RunStringの違いは、変換元にメモリ上の文字列を使用するか
　　変換元のファイルを指定するかになります。


　3.テンプレートのクリンナップ
　　M2H_CleanupTemplate関数を呼び出して、テンプレートを開放します。

　・生成物について
　　生成物の著作権、つまりこのツールによって
　　変換されたファイルの著作権は、元のファイルの著作者に在るものとします。

　　ただし、テンプレート等に含まれるライセンス類は
　　記述された各種ライセンスに従って処理するようにお願いいたします。


●テンプレートのライセンスについて
　template以下に存在するファイルは、KGIndirectWriterのライセンスに従うものとします。
　具体的なライセンスはtemplate/footer.htmlに記述されているライセンス表示
　または同梱のKGIndirectWriter.license.txtをご確認ください。

　未変更のテンプレートによって出力された場合は、当該出力ファイルにライセンス表記が含まれます。
